# 贪吃蛇游戏

这是一个简单的贪吃蛇游戏项目，使用HTML、CSS和JavaScript构建。玩家可以通过键盘控制蛇的移动，吃掉食物以获得分数。

## 项目结构

```
snake-game
├── src
│   ├── index.html        # 游戏的主HTML文件
│   ├── styles
│   │   └── style.css     # 游戏的样式定义
│   ├── scripts
│   │   └── app.js        # 游戏的主要逻辑
├── package.json          # npm的配置文件
└── README.md             # 项目的文档
```

## 如何运行

1. 克隆这个仓库到本地：
   ```
   git clone <repository-url>
   ```

2. 进入项目目录：
   ```
   cd snake-game
   ```

3. 安装依赖：
   ```
   npm install
   ```

4. 打开 `src/index.html` 文件在浏览器中运行游戏。

## 游戏规则

- 使用箭头键控制蛇的移动。
- 吃掉食物以增加分数和蛇的长度。
- 避免撞墙或撞到自己。

祝你玩得开心！
